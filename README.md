## 💡 Conseils

- **LMDW = distributeur exclusif Signatory en France** → catalogue le plus exhaustif
- Vérifier les scores sur [Whiskybase](https://www.whiskybase.com) avant d'acheter
- **Cask Strength + non chillfiltered** = meilleure intégrité du whisky
- Pour comparer : [The Whisky Exchange](https://www.thewhiskyexchange.com/brands/signatory-vintage)

## 🔄 Mise à jour automatique

Le script `update.py` tourne chaque **lundi à 7h** via GitHub Actions.  
Il vérifie les disponibilités et met à jour `data/bottles.json`.

Pour forcer une mise à jour manuelle : **Actions → Weekly Signatory Update → Run workflow**
